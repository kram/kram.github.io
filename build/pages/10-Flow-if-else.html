Control flow

If / Else

if and else are currently the only available control structures implemented in kram.

if condition {
    // Was true
} else {
    // Not true
}
if condition {
    // Was true
}

While

kram only has a simple way of looping (for now), the while-loop.

Example

// Prints the numbers 1 to 10
i := 1

while i <= 10 {
    IO::Println(i)
    i := 1
}
