# Types / Number

----

A `Number` is the only way to represent a number (eg, 10, 1337 or 123.456) in kram.

```go
var positive = 1337
var price = 999.95
```

## Methods

*More methods will be added when needed, see https://golang.org/pkg/math for ideas.*

### `Number.Sqrt() Number`

```go
IO::Println(100.Sqrt()) // 10
```
